Командная строка

Командная строка Windows позволяет выполнять различные задачи с помощью команд. Примеры команд:

  • dir - отображает список файлов и папок в текущем каталоге.
  • cd - изменяет текущий каталог.
  • copy - копирует файлы.

Как использовать справку

Для получения справки о команде, введите команду в командной строке:

  • help - выводит список всех доступных команд.
  • help [команда] - показывает справку по конкретной команде.
